Архитектурный линтер для Go и Rust проектов

1
Создайте issue в GitHub github.com/mshogin/archlint Заголовок: scan: ваш-username/ваш-repo
2
Бот проанализирует ваш проект Результат появится в комментарии к issue
3
Получите health score и список нарушений Слои архитектуры, запрещённые зависимости, coupling

Добавить в CI
# .github/workflows/archlint.yml
name: archlint
on: [push, pull_request]
jobs:
  lint:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: mshogin/archlint-action@v1
